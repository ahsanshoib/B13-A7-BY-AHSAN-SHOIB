# KeenKeeper

KeenKeeper is a simple and smart friendship management web application. It helps you stay connected with your friends and keep track of your social interactions easily.

# Technologies Used

* React.js - For building the user interface.
* Tailwind CSS - For modern and clean styling.
* Lucide React - For beautiful icons.
* Recharts - For interactive data visualization.

#  Key Features

1. Friendship Analytics: View your interaction history (Calls, Texts, Videos) through an interactive Pie Chart.
2. Dynamic Dashboard: Track your total friends, active connections, and friends who need attention in real-time.
3. Interactive Timeline: Log every interaction and filter your friendship history to see how often you connect.
